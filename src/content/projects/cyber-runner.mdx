# Cyber Runner

An endless runner game built entirely in the browser using React Three Fiber.

## Tech Stack
- **Physics:** Rapier.js for deterministic physics.
- **State:** Zustand for high-frequency game loop management.
- **Shaders:** Custom GLSL for the neon city generation.
